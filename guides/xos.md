## 🚀 Flashing Process
1. 1️⃣ **Reboot to Recovery** (TWRP / OrangeFox)
2. 2️⃣ **Flash Vbmeta Disabler** zip file
3. 3️⃣ **Flash ROM.zip** (XOS 16.1 Port)
4. 4️⃣ **Format Data** (Select wipe -> format data, type `yes`)
5. 5️⃣ **Reboot System**
